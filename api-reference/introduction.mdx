# API Reference

The Cortana AI Public API provides programmatic access to contacts, conversations, messages, agents, appointments, conversions, tracking sessions, tags, custom fields, financial data (Stripe, Shopify, Whop), voice call logs/transcripts, and **creating businesses** for automated agency onboarding.

## Base URL

Most resources live under a business:

```
https://app.usecortana.ai/api/v1/businesses/{businessId}
```

**Create business** (agency-level, no existing `businessId`):

```
POST https://app.usecortana.ai/api/v1/businesses
```

**List business templates** (saved with *Copy as template* in the Control Centre — use an `id` as `templateId` when creating):

```
GET https://app.usecortana.ai/api/v1/businesses/templates
```

All endpoints require Bearer token authentication.

## Authentication

```bash
Authorization: Bearer sk-ak-your-api-key
```

See [Authentication](/authentication) for creating and managing API keys.

## Response Format

```json
{
  "data": [...],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 150,
    "totalPages": 8
  }
}
```
