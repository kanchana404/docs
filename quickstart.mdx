# Quickstart

The fastest way to start is the MCP server — connect Claude or Cursor to your data in 2 minutes.

## Option 1: MCP Server (AI tools)

<Steps>
  <Step title="Install with one command">
    ```bash
    npx -y @cortana-ai/mcp-server@latest
    ```
  </Step>
  <Step title="Add to your AI tool">
    See the [MCP Setup Guide](/mcp/setup) for Claude Desktop, Claude Code, Cursor, and VS Code configs.
  </Step>
  <Step title="Start asking questions">
    "What's my MRR?" / "Which ad has the best ROAS?" / "Show me churned customers"
  </Step>
</Steps>

## Option 2: REST API (programmatic)

<Steps>
  <Step title="Get your API key">
    Go to **Settings → API Keys → Generate New Key** in your dashboard.
  </Step>
  <Step title="Make your first request">
    ```bash
    curl -H "Authorization: Bearer sk-ak-your-key" \
      https://app.usecortana.ai/api/v1/businesses/{businessId}/contacts
    ```
  </Step>
  <Step title="Explore the API">
    Browse the [API Reference](/api-reference/introduction) for all endpoints. Agencies automating client setup can use [Create business](/api-reference/endpoint/create-business) (`POST /api/v1/businesses` with the `businesses:write` scope).
  </Step>
</Steps>
