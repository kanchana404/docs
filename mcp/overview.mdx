# MCP Server

The Cortana AI MCP (Model Context Protocol) server gives AI tools full read access to your business data. Ask questions in natural language — the AI picks the right tools automatically.

## What is MCP?

MCP is an open protocol that lets AI assistants connect to external data sources. Instead of copy-pasting data into Claude, the AI can directly query your attribution, Stripe, Shopify, conversations, and CRM data.
