# Authentication

All API requests and the MCP server require a Bearer token. Most API keys are used with **per-business** routes (`/api/v1/businesses/{businessId}/...`) and configurable scopes. **Creating a new business** uses `POST /api/v1/businesses`, and **listing agency business templates** (for `templateId`) uses `GET /api/v1/businesses/templates` — both use the same path prefix without `{businessId}` and require the **`businesses:write`** scope — use the **Agency / onboarding automation** preset or add that scope manually; it is **not** included in the default Chrome Extension preset.

## Create an API Key

<Steps>
  <Step title="Open Settings">
    Navigate to **Settings → API Keys** in your Cortana AI dashboard.
  </Step>
  <Step title="Generate New Key">
    Click **Generate New Key**. Give it a name (e.g., "MCP Server" or "Dashboard").
  </Step>
  <Step title="Select Scopes">
    Enable the scopes you need. For full MCP access, enable all read scopes.
  </Step>
  <Step title="Copy the Key">
    Your key starts with `sk-ak-`. Copy it immediately — it won't be shown again.
  </Step>
</Steps>

## Using the API Key

```bash
curl -H "Authorization: Bearer sk-ak-your-api-key" \
  https://app.usecortana.ai/api/v1/businesses/{businessId}/contacts
```

## Available Scopes

| Scope | Access |
|-------|--------|
| `contacts:read` | List and search contacts |
| `contacts:write` | Create, update, delete contacts |
| `conversations:read` | List conversations, read messages, call logs |
| `conversations:write` | Update conversation metadata |
| `messages:read` | Read message history |
| `messages:write` | Send messages |
| `agents:read` | List and view agent configurations |
| `agents:write` | Create, update, delete agents |
| `appointments:read` | List and view appointments |
| `appointments:write` | Create, update, delete appointments |
| `conversions:read` | View conversion entries and configs |
| `conversions:write` | Create conversion entries |
| `tracking:read` | View tracking sessions |
| `tags:read` / `tags:write` | Manage tags |
| `custom_fields:read` / `custom_fields:write` | Manage custom fields |
| `attribution:read` | Full attribution table, UTMs, presets, LTV |
| `stripe:read` | Stripe payments, subscriptions, customers, metrics |
| `shopify:read` | Shopify orders, customers, products, analytics |
| `whop:read` | Whop payments, customers, memberships |
| `meetings:read` | Read meetings / Fathom data |
| `businesses:write` | **Create businesses** and **list agency business templates** (`POST /api/v1/businesses`, `GET /api/v1/businesses/templates`) — agency owner, superuser, or admin manager only |
